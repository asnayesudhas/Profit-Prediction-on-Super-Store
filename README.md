Profit-Prediction-on-Super-Store
A Streamlit-based machine learning app that predicts Superstore profit using Category, Sub-Category, Sales, Quantity, and Discount as inputs. Built with Python and Gradient Boosting Regressor to support data-driven retail profitability analysis.

Superstore Profit Prediction App:

This project is a **machine learning–based Streamlit web application** that predicts the **profit** of a retail order using key business features. It helps understand how sales, discounts, and product categories influence profitability.

Features:

* Interactive Streamlit UI
* User inputs:

  * Category
  * Sub-Category
  * Sales
  * Quantity
  * Discount
* Predicts Profit
* Uses **Gradient Boosting Regressor** for prediction
* Data preprocessing with encoding and scaling

Technologies Used:

* Python
* Streamlit
* Pandas & NumPy
* Scikit-learn

Dataset:

* Superstore Sales Dataset
* Contains retail transaction details such as category, sales, discount, and profit.
  
How to Run the Project:

1. Clone the repository

```bash
git clone https://github.com/your-username/superstore-profit-prediction.git
```

2. Install required libraries

```bash
pip install streamlit pandas scikit-learn
```

3. Run the Streamlit app

```bash
streamlit run streamlit_profit_app.py
```
Use Case:

* Retail profit analysis
* Understanding discount impact
* Data science mini projects
* Academic and learning purposes

Output:

The application predicts the **expected profit value** based on the user’s input.
